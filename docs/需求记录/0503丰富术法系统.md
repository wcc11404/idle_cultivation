## 3. 战斗结算口径

## 3.1 伤害计算公式

基础伤害改为：

```text
伤害 = max(攻击 * (1 - max(防御-穿透,0) / max(防御-穿透+K, K)), 1) * 系数
```

解释：
- `攻击`：攻击方最终攻击
- `防御`：防守方最终防御
- `穿透`：攻击方最终穿透
- `K`：防御折算常数，当前固定取 `100`
- `系数`：主动术法或普通攻击对应的伤害倍率
- 最终伤害最低保底为 `1`

## 3.2 命中判定

```text
是否命中 = 判定（命中 - 闪避）
```

当前实现口径：
- 我方与敌方初始命中率都为 `100%`
- 命中判定使用：`我方命中率 - 敌方闪避率`
- 用该结果与随机数比较决定是否命中
- 若未命中，则本次伤害直接为 `0`

## 3.3 暴击判定

```text
是否暴击 = 判定（暴击率 - 敌方抗暴率）
```

当前实现口径：
- 我方与敌方初始暴击率、抗暴率都为 `0%`
- 暴击判定使用：`我方暴击率 - 敌方抗暴率`
- 用该结果与随机数比较决定是否暴击

## 3.4 暴击伤害

```text
触发暴击后的伤害 = 伤害 * (1 + 爆伤)
```

解释：
- `爆伤` 是额外伤害倍率，不是最终倍数本身
- 例如：`爆伤 = 100%` 时，暴击后伤害为 `2.0 倍`

## 4. 术法系统总览

## 4.1 术法分类

术法分为 4 大类：

- 吐纳
- 攻击
- 开局
- 生产

说明：
- 吐纳类：主要作用于修炼或资源积累相关机制
- 攻击类：主要作用于战斗中的主动伤害与附加效果
- 开局类：战斗开始时直接生效的属性或状态加成
- 生产类：当前只是预留类型，后续可用于炼丹、采集、制造等生产玩法增强

## 4.2 五行属性

每个术法都带五行属性之一：

- 金
- 木
- 水
- 火
- 土
- 无

说明：
- “无” 表示无属性术法
- 当前文档只确定术法自带五行标签，五行相克或五行联动效果暂未展开

## 4.3 稀有度

每个术法都带稀有度：

- 凡
- 黄
- 玄
- 地
- 天

当前约束：
- 稀有度只区分掉落概率与获取难度
- 稀有度本身不直接决定最终强度上限
- 真正强度由模板参数、等级、星级、属性加成、技能效果共同决定

## 4.4 等级与星级

每个术法包含：
- 等级：`1-10`
- 星级：`0-5`

其中：
- 等级提升：复用现有升级条件机制
  - 使用次数
  - 灵气消耗
  - 其他现有升级条件
- 星级提升：新增独立升星机制

## 4.5 升级与升星的区别

### 升级

升级会提升：
- 属性加成
- 技能效果

### 升星

升星只提升：
- 属性加成

升星不会提升：
- 技能效果

### 升星消耗

升星需要：
- 对应术法解锁道具
- 升星材料 `空白玉简`

当前规则：
- 升几星就需要几个对应术法解锁道具
  - 升 `1星` 需要 `1` 个
  - 升 `2星` 需要 `2` 个
  - 升 `3星` 需要 `3` 个
  - 依此类推
- `空白玉简` 从 `2星 -> 3星` 开始，每次升星固定需要 `1` 个

## 5. 术法模板参数

## 5.1 吐纳类

吐纳类术法：
- 不走统一伤害模板
- 全部是特殊机制型效果

例如：
- 灵气逸散概率变化
- 丹毒减少
- 恢复增强

## 5.2 攻击类模板

攻击类术法基础倍率按稀有度划分：

- 凡：`1.1`
- 黄：`1.3`
- 玄：`1.6`，附带 `1` 个特殊机制
- 地：`2.0`，附带 `1` 个特殊机制
- 天：`2.5`，附带 `1-2` 个特殊机制

等级成长：
- 每升一级：`+5%` 伤害
- 最后一级：`+10%` 伤害

当前实现口径：
- `1->2` 到 `8->9` 都按每级 `+5%`
- `9->10` 按 `+10%`
