# 经脉系统设计文档

## 一、系统概述

经脉系统是筑基期解锁的永久属性成长系统，通过充灵气+消耗材料打通穴位，获得静态最终属性加成。

### 前置条件
- **境界要求**：筑基期解锁
- **解锁触发**：突破筑基后自动触发新手引导

## 三、核心玩法流程

```
┌─────────────────────────────────────────────────────────┐
│  穴位状态：未激活 → 充灵气中 → 可突破 → 已打通          │
└─────────────────────────────────────────────────────────┘

步骤1: 长按经脉区域 → 开始充灵气
步骤2: 持续按住，充灵速度递增，消耗玩家灵气（1:1）
步骤3: 灵气充满 → 穴位变为"可突破"
步骤4: 点击穴位 → 弹窗显示详情 → 消耗材料突破 → 100%成功
步骤5: 穴位点亮 → 获得属性加成
```

## 四、充灵气机制

### 交互方式
- **触发区域**：经脉中央区域任意位置（穴位除外）
- **触发方式**：长按约0.5秒后开始充灵
- **持续按住**：充灵速度随时间递增
- **灵气消耗**：充灵数值:消耗数值 = 1:1

### 充灵速度递增
```
初始速度：10点/秒
速度增长：每秒增加10点

时间线示例：
第1秒：10点/秒（累计10点）
第2秒：20点/秒（累计30点）
第3秒：30点/秒（累计60点）
第4秒：40点/秒（累计100点）
第5秒：50点/秒（累计150点）
...
第10秒：100点/秒（累计550点）
```

### 停止充灵
- 玩家松手 → 立即停止
- 玩家灵气耗尽 → 自动停止
- 充能达到穴位上限 → 自动停止

### 穴位灵气容量
```
穴位N所需灵气 = 基础值 × 1.5^(N-1)

示例（某经脉，基础值100）：
第1穴：100
第2穴：150
第3穴：225
第4穴：338
第5穴：506
第6穴：759
第7穴：1139
第8穴：1709
```

## 五、穴位交互

### 点击穴位弹窗内容

```
┌─────────────────────────────────────┐
│         【第3穴 - 中府穴】          │
│                                     │
│  状态：充灵中 (150/225)             │
│                                     │
│  ─────── 突破条件 ───────          │
│  灵气：225 点                       │
│  丹药：聚气丹 ×1                    │
│  材料：灵石 ×100                    │
│                                     │
│  ─────── 属性加成 ───────          │
│  气血上限 +50                       │
│                                     │
│  ─────── 拥有材料 ───────          │
│  聚气丹：5/1 ✓                      │
│  灵石：12580/100 ✓                  │
│                                     │
│      [取消]        [突破]           │
└─────────────────────────────────────┘
```

### 弹窗状态
- **未充满**：显示充灵进度，突破按钮置灰
- **已充满但材料不足**：材料显示红色，突破按钮置灰
- **已充满且材料足够**：突破按钮可点击
