# 数值公式
伤害 = max(攻击 * (1 - max(防御-穿透,0) / max(防御-穿透+K, K)),1) * 系数
是否命中 = 判定（命中-闪避）
是否暴击 = 判定（暴击-抗暴）
触发暴击后的伤害 = 伤害 * (1+爆伤)

# 术法 
- 概述
术法区分 吐纳，攻击，开局，生产4种类型，每种术法区分五行属性，金木水火土无，每种术法区分稀有度，凡黄玄地天，
每种术法有等级1-10，星级0-5，等级突破需要达成升级条件（使用次数，灵气等，复用现有配置），星级突破需要消耗特定数量的术法解锁道具（升到1星需要一个，升到5星需要5个） + 升星材料（道具叫空白玉简，3星到5星每次升星需要1个）
升级增加属性加成和技能效果，升星只增加属性加成，稀有度只区分掉落概率，和强度无实际关系

- 模板参数
吐纳类，全是特殊机制
伤害类，凡1.1，黄1.3，玄1.6（附带一个特殊机制），地2.0（附带一个特殊机制），天2.5（附带1-2个特殊机制），每升一级+5%伤害，最后一级+10%伤害
